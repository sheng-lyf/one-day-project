# 玄学命理网

一个功能齐全的在线命理咨询平台，集成了传统中国命理学与现代网页技术。

## ✨ 功能特色

### 🎯 八字排盘
- 基于出生日期和时间进行精确的八字分析
- 支持性别区分的命盘计算
- 详细的五行分析和解读

### 📝 姓名学分析
- 中文姓名的笔画分析
- 五格三派姓名学理论
- 个性化姓名建议和解读

### 🏞️ 风水方位计算
- 个人风水吉位计算
- 基于出生年份和性别的方位分析
- 生活布局建议

### ⭐ 星座运势
- 十二星座详细信息
- 个性化运势预测
- 综合运势分析

## 🚀 技术特点

- **响应式设计** - 完美适配各种设备
- **纯前端实现** - 无需服务器，所有数据本地处理
- **用户友好** - 简洁直观的操作界面
- **隐私保护** - 用户数据仅在本地处理

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript (ES6+)** - 交互逻辑
- **Google Fonts** - 字体支持

## 🎮 使用方法

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 选择需要的功能模块
4. 输入相应信息进行测算

## 🔧 功能模块

### 八字排盘
输入出生日期、时间和性别，系统将自动计算八字排盘并提供专业解读。

### 姓名学分析
输入中文姓名，系统将分析姓名的笔画、五行属性并提供详细解读。

### 风水计算
根据出生年份和性别，计算个人的风水吉位和方位建议。

### 星座运势
点击对应的星座图标，即可查看该星座的详细运势分析。
